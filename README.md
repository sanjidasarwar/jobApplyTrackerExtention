🚀 Job Apply Tracker – Chrome Extension
A simple and powerful Chrome Extension to track your job applications directly from job sites like LinkedIn, Glassdoor, and more.

✨ Features - 📌 Save job applications with one click - 🏷 Track application status (Applied, Interview, Rejected, Offer) - 🔗 Automatically capture job title and URL - 💾 Store data locally (no login required) - ⚡ Lightweight and fast

🧠 Problem It Solves

Applying to jobs across multiple platforms can get messy.
You lose track of:

    - Where you applied
    - Application status
    - Important job links

Job Apply Tracker solves this by keeping everything in one place.

🛠 Tech Stack - JavaScript (Vanilla) - Chrome Extension API (Manifest V3) - HTML + CSS - Chrome Storage API

📦 Installation 1. Clone this repository:
git clone https://github.com/sanjidasarwar/jobApplyTrackerExtention.git 2. Open Chrome and go to:
chrome://extensions/ 3. Enable Developer Mode 4. Click Load unpacked 5. Select the project folder

🚀 Usage 1. Go to any job listing page (LinkedIn, Glassdoor, etc.) 2. Click the extension icon 3. Click "Save This Job" 4. View and manage saved jobs from the popup
